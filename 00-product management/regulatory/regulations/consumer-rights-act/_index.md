# Consumer Rights Act 2015

## Overview
| Field | Value |
|---|---|
| Full Name | Consumer Rights Act 2015 |
| Jurisdiction | United Kingdom |
| Effective | 1 October 2015 |
| Enforcing Bodies | CMA (Competition and Markets Authority); Trading Standards; FCA (for financial services) |
| Penalties | Court orders; injunctions; damages; CMA enforcement |
| Scope | Contracts between traders and consumers for goods, services, and digital content |

## Key Parts for Banking IT
| Part | Title | Banking Relevance |
|---|---|---|
| Part 1 | Consumer contracts for goods | Physical items (cards, tokens, devices) |
| Part 1 Ch 3 | Digital content | Mobile banking app; online banking platform |
| Part 1 Ch 4 | Services | Banking services (accounts, transfers, advice) |
| Part 2 | Unfair terms | Terms and conditions; contract terms |

## Key Provisions

### Digital Content Rights (Ch 3, s33-47)
| Right | Description | IT Implication |
|---|---|---|
| Satisfactory quality | Digital content must be of satisfactory quality | Banking app must work reliably; no critical bugs |
| Fit for purpose | Must be fit for any particular purpose communicated | Features must work as described |
| As described | Must match any description given | UI must match marketing/documentation |
| Right to repair or replacement | If digital content is faulty | Bug fixes; app updates; alternative access |
| Right to price reduction | If repair/replacement not possible | Service credits; fee waivers |

### Services Rights (Ch 4, s48-57)
| Right | Description | IT Implication |
|---|---|---|
| Reasonable care and skill | Services performed with reasonable care and skill | Systems must be reliable; adequate testing |
| Information as term | Pre-contract information becomes a term | Advertised features must be delivered |
| Reasonable price | If price not agreed, must be reasonable | Fee transparency |
| Reasonable time | If time not agreed, must be within reasonable time | SLA for service delivery |

### Unfair Terms (Part 2, s62-69)
| Provision | Description | IT Implication |
|---|---|---|
| Fairness test | Term is unfair if it causes significant imbalance to consumer's detriment | T&C must be balanced; no one-sided clauses |
| Transparency | Terms must be in plain, intelligible language | T&C in plain language; key terms highlighted |
| Prominence | Key terms must be prominent | Important terms not buried in small print |
| Grey list (Schedule 2) | Indicative list of potentially unfair terms | Review T&C against grey list |
